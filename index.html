Javascript : Definition
JavaScript® (often shortened to JS) is a lightweight, interpreted, object-oriented language with first-class functions, most known as the scripting language for Web pages.​
Javascript : Usage in Front End Development​
  • DOM Manipulation, Adding user interactivity to web pages.​
  • Dropdown, Slider/Carousel, Parallax, AJAX, you name it.

Javascript : jQuery​
jQuery is the most popular JavaScript library designed to simplify the client-side scripting of HTML
Javascript : Why use jQuery?​
  • Faster development.
  • Library dependencies (lots of Javascript Libraries rely on jQuery)​

Example (Pure Javascript) :

var el = document.querySelector('.bye');​
el.style.background = "red";​
el.style.color = "white";​

Example (jQuery) :

var el = $('.bye');​
el.css( { 'background' : 'red' , 'color' : 'white' } );

(Example) Select some html elements

});​
